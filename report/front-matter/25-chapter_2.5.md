2.5. Ubiquitous Language 

| Término | Definición | Términos relacionados |
|--------|-----------|----------------------|
| Usuario (User) | Profesional de empresas prestadoras de servicios de agua o gestión de residuos que utiliza Aquanetix para monitorear y gestionar la red hidráulica. | Operador, Supervisor, Perfil de usuario |
| Operador (Operator) | Usuario encargado de supervisar los datos en tiempo real y gestionar alertas dentro del sistema. | Monitoreo, Alerta, Sensor |
| Supervisor (Supervisor) | Usuario responsable de analizar la información del sistema y tomar decisiones operativas y estratégicas. | Reporte, Dashboard, Mantenimiento |
| Sensor IoT (IoT Sensor) | Dispositivo instalado en la red hidráulica que mide variables como pH, flujo y gases. | Datos del sensor, Monitoreo, Parámetros del agua |
| Monitoreo (Monitoring) | Proceso de seguimiento continuo de los parámetros del agua en tiempo real. | Datos del sensor, Dashboard, Red hidráulica |
| Alerta (Alert) | Notificación generada cuando se detecta una anomalía en los parámetros del sistema. | Anomalía, Umbral, Incidencia |
| Anomalía (Anomaly) | Variación fuera de los valores normales en los parámetros del agua. | Alerta, Monitoreo, Zona crítica |
| Mantenimiento (Maintenance) | Acciones realizadas para prevenir o corregir fallas en la red hidráulica. | Zona crítica, Incidencia, Planificación |
| Agua Reutilizable (Reusable Water) | Agua residual que cumple con los parámetros necesarios para su reutilización. | Validación de reutilización, Parámetros del agua |
| Validación de Reutilización (Reuse Validation) | Proceso de verificación de que el agua cumple con las condiciones para ser reutilizada. | Agua reutilizable, Monitoreo, Parámetros |
| Asignación de Agua (Water Allocation) | Distribución del agua reutilizada hacia procesos o sectores que la requieren. | Agua reutilizable, Logística, Reutilización |
| Plataforma Aquanetix (Platform) | Sistema web que integra sensores IoT para monitorear, analizar y gestionar la red hidráulica y la reutilización del agua. | Dashboard, Monitoreo, Alertas |
